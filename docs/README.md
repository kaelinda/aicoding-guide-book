# AI Coding Guide Book 📚

> 面向初级 iOS 开发者的 AI 编程实践指南
> 从环境搭建到多 Agent 协作，系统掌握 AI 辅助开发的核心方法

---

## 🎯 这是什么

一份围绕 **AI 辅助编程** 的系统化教程，面向：

- 有基础 Swift / Xcode 经验的 iOS 开发者
- 想把 AI 真正用进日常开发流程的独立开发者
- 对 Claude Code、Cursor、Codex、MCP、Agent 协作感兴趣的学习者

不是单纯的工具说明书，而是从 **入门体验 → 核心概念 → 实战方法 → 高阶工作流** 的完整学习路径。

## ✨ 本书特色

- **面向真实开发**：围绕工程问题展开，不是空谈概念
- **面向初级开发者**：不用先成为 AI 专家，从一行命令开始
- **强调“生成之后的判断力”**：AI 能生成很多代码，重点是你能不能判断、修改、验收
- **关注长期工作流**：Prompt 组织、代码验收、安全边界、多工具协作、自动化
