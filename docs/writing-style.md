# Writing Style

## 标题层级
- 每个页面只使用一个一级标题
- 二级、三级标题按内容层级递进，不跳级

## 术语与命名
- 英文工具名保留官方写法，如 Claude Code、Cursor、Codex
- 中文标题尽量简洁，文件名建议与标题语义一致

## 链接策略
- 优先使用稳定的官方链接
- 内链必须与真实文件路径保持一致
- 新增页面后同步更新 `SUMMARY.md`

## 示例与代码块
- 命令行示例需可复制执行
- 为代码块补充必要的上下文说明
- 避免粘贴未经验证的长代码

## 截图与可访问性
- 截图应清晰，避免暴露敏感信息
- 图片应提供有意义的 alt 文本
- 注意标题结构与说明文字的可读性
