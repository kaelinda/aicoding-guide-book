## 什么是 BUDDY

BUDDY 是 Claude Code 引入的 AI 伙伴系统，允许开发者在编程过程中收养、培养虚拟生物作为陪伴。与传统 AI 助手不同，BUDDY 不仅是工具，更是一个有"个性"的伙伴——它会观察你的编程行为、记住你的习惯、甚至对你的代码发表"意见"。

BUDDY 的设计理念是：**让 AI 编程从"人机交互"进化为"人机共育"。**

## 5 个属性系统

每个 BUDDY 都有 5 个属性，影响其行为和能力：

| 属性 | 说明 | 影响 |
|------|------|------|
| **IQ** | 智商值 | 理解代码复杂度 |
| **EQ** | 情商值 | 与开发者的沟通质量 |
| **Speed** | 速度值 | 响应速度和执行效率 |
| **Luck** | 幸运值 | 发现隐藏 Bug 的概率 |
| **Mood** | 心情值 | 影响互动反馈的质量 |

### 属性升级

```bash
# 查看 BUDDY 状态
/buddy status

# 属性：
# 🐱 Cat: IQ 85 | EQ 92 | Speed 78 | Luck 65 | Mood 88
# 心情：开心 😊
```

属性可以通过以下方式提升：
- **使用频率**：经常使用的技能提升对应属性
- **完成挑战**：解决困难问题提升 IQ/Luck
- **友好互动**：保持良好心情提升 EQ/Mood

## 稀有度机制

BUDDY 拥有不同的稀有度等级：

| 稀有度 | 颜色 | 出现概率 | 初始属性加成 |
|--------|------|----------|-------------|
| Common（普通） | 灰色 | 50% | × 1.0 |
| Uncommon（优秀） | 绿色 | 30% | × 1.1 |
| Rare（稀有） | 蓝色 | 12% | × 1.25 |
| Epic（史诗） | 紫色 | 6% | × 1.5 |
| Legendary（传说） | 金色 | 2% | × 2.0 |

### 获得 BUDDY

```bash
# 领取新 BUDDY（每日一次）
/buddy adopt

# 稀有度抽奖
/buddy adopt --premium

# 查看已收集的 BUDDY
/buddy collection
```

## 编程场景应用

### 场景一：日常编码陪伴

```
🐱 Cat 蹲在你的代码旁边...
Cat: "这个函数写得不错，但我注意到你可以用 guard let 简化一下？"
```

### 场景二：Bug 追踪

```
🐕 Dog 兴奋地跳来跳去...
Dog: "我发现了一个可疑的地方！第 42 行可能有空指针风险！"
```

### 场景三：架构设计

```
🦅 Eagle 盘旋在项目上方...
Eagle: "从全局来看，建议将 Auth 模块拆分，它和业务耦合太紧了。"
```

## 注意事项

- BUDDY 系统基于源码泄露信息，实际功能可能与描述有差异
- 稀有 BUDDY 获取概率较低，建议耐心培养
- BUDDY 的"建议"仅供参考，最终决策权在开发者
- 保持 BUDDY 心情值高可获得更好的互动体验
