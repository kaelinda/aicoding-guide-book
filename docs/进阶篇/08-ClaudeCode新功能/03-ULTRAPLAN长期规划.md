## 什么是 ULTRAPLAN

ULTRAPLAN 是 Claude Code 推出的云端深度规划服务，专门用于处理需要 30 分钟以上的复杂技术决策。与本地快速分析不同，ULTRAPLAN 将任务上传到云端，利用云端更强的算力和更长的上下文窗口，进行深度思考和全面分析。

典型适用场景：
- **系统架构设计**：需要全面分析现有系统，制定重构计划
- **技术选型评估**：评估多个技术方案的优劣和风险
- **技术路线图**：制定 3-6 个月的技术演进计划
- **遗留系统改造**：分析遗留代码，制定迁移策略

ULTRAPLAN 的核心理念是：**把 AI 从"快速助手"升级为"战略顾问"。**

## 工作机制

### 本地发起

```bash
# 启动 ULTRAPLAN
/ultraplan start

# 指定规划主题
/ultraplan start "重构 iOS 项目架构，从 MVC 迁移到 MVVM"

# 查看规划状态
/ultraplan status

# 取消规划
/ultraplan cancel
```

### 云端处理

```
[本地] 任务上传中...
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📤 上传内容：
├── 项目代码结构（分析后）
├── 现有架构文档
├── 团队技术栈
└── 规划要求
━━━━━━━━━━━━━━━━━━━━━━━━━━━

[云端] 启动深度分析...
⏱️ 预计耗时：30 分钟
🔗 进度追踪：https://claude.ai/ultraplan/xxx
```

### 规划过程

ULTRAPLAN 在云端会执行以下分析：

1. **现状扫描**：理解现有代码库的结构和质量
2. **目标拆解**：将最终目标分解为可执行的阶段
3. **风险评估**：识别迁移过程中的风险点
4. **方案设计**：制定详细的技术方案和执行步骤
5. **资源估算**：评估所需时间和人力
6. **备选方案**：提供多个可行方案及优劣对比

### 结果交付

```bash
[云端] 规划完成！
📬 规划报告已发送到你的邮箱

# 本地查看
/ultraplan view

# 下载完整报告
/ultraplan download --format markdown
```

## 浏览器审批界面

### 访问方式

规划完成后，你会收到一封邮件和一个链接：

```
📧 邮件标题：[Claude Code] ULTRAPLAN 规划完成

🔗 点击查看：
https://claude.ai/ultraplan/review/xxx

或直接在 Claude Code 中：
/ultraplan review
```

### 界面功能

浏览器审批界面提供以下功能：

| 功能 | 说明 |
|------|------|
| 📄 完整报告 | 查看规划报告全文 |
| 📊 可视化时间线 | 图形化展示执行计划 |
| ✅ 逐项审批 | 对每个阶段单独审批 |
| 💬 反馈评论 | 对规划提出修改意见 |
| 📥 导出 | 导出为 Markdown/JSON/PDF |
| 🚀 一键执行 | 审批后自动开始执行 |

### 审批流程

```
Step 1: 查看摘要
├── 规划目标：MVC → MVVM 重构
├── 总耗时：预计 8 周
├── 风险等级：Medium
└── 资源需求：2 人

Step 2: 查看详细方案
├── Phase 1（第 1-2 周）：分析现有代码
├── Phase 2（第 3-5 周）：逐步迁移 ViewController
├── Phase 3（第 6-7 周）：重构 Model 层
└── Phase 4（第 8 周）：测试与上线

Step 3: 审批决策
├── [✅ 批准] 开始执行
├── [💬 评论] 提出修改意见
└── [📥 保存] 先保存，稍后决定
```

```
ULTRAPLAN 输出：
├── 代码分析报告：模块依赖图

├── 分阶段计划：6 个月
├── 风险评估：数据迁移、停机窗口
└── 测试策略：如何保证零停机
```

## 配置选项

```json
// .claude/ultraplan.json
{
  "enabled": true,
  "timeout": 60,          // 最大规划时间（分钟）
  "contextDepth": "deep", // deep | standard
  "notification": {
    "email": true,
    "slack": false
  },
  "autoReview": {
    "enabled": false,     // 自动批准低风险规划
    "riskThreshold": "low"
  }
}
```

## 注意事项

- **隐私**：上传到云端前，建议在本地先过滤敏感信息
- **成本**：ULTRAPLAN 是付费功能，每次规划消耗相应的 Credits
- **等待时间**：复杂规划可能需要 30 分钟以上，请耐心等待
- **审批重要性**：不建议开启自动审批，确保每个规划都经过人工审核
