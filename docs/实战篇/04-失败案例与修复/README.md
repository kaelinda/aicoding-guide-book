## 为什么 AI 会生成错误代码？

### 1. 上下文理解偏差
AI 对需求的理解可能与你的预期不同。

### 2. 边界情况遗漏
AI 倾向于生成"正常路径"代码，忽略边界情况。

### 3. 隐含假设
有些"常识"AI 不知道，需要明确说明。

### 4. 训练数据过期
AI 的知识有截止日期，可能不知道最新的 API。

---

## 学习方法

### 1. 先运行，再分析

先让代码跑起来，观察实际行为。

### 2. 对比预期 vs 实际

明确区分：
- 预期行为：你的设计意图
- 实际行为：代码的实际表现

### 3. 隔离问题

逐步缩小问题范围：
```
整体功能 → 具体页面 → 具体函数 → 具体逻辑
```

### 4. 小步验证

每次修改后立即验证，不要一次性改太多。
