# 01-Cursor 核心功能

> 学 Cursor 只记四个快捷键：`Cmd+L` / `Cmd+K` / `Tab` / `Cmd+Shift+L`。其余都是变奏。

---

## 1. AI 对话（Chat）— `Cmd + L`

**适合**：询问代码问题、生成新代码、解释复杂逻辑、代码审查。

```text
你：创建一个 SwiftUI 视图，显示用户列表，每个单元格显示头像和用户名。

Cursor：
struct UserListView: View {
    let users: [User]

    var body: some View {
        List(users) { user in
            HStack {
                AsyncImage(url: user.avatarURL) { image in
                    image.resizable()
                } placeholder: {
                    ProgressView()
                }
                .frame(width: 40, height: 40)
                .clipShape(Circle())

                Text(user.name)
                    .font(.headline)
            }
        }
    }
}
```

**小技巧**：

- 对话会自动引用当前打开的文件。你可以用 `@` 符号显式指向其他文件（`@UserService.swift`）
- 复杂问题先问规划再让它动手：`先不要写代码，给我 3 种实现思路对比`

---

## 2. 内联编辑（Inline Edit）— `Cmd + K`

**适合**：快速修改选中的代码、重构函数、加注释、优化性能。

```text
选中代码 → Cmd + K → 输入指令

原代码：
func calculateTotal(items: [Item]) -> Double {
    var total = 0.0
    for item in items {
        total += item.price
    }
    return total
}

你：用 reduce 重写

Cursor：
func calculateTotal(items: [Item]) -> Double {
    items.reduce(0) { $0 + $1.price }
}
```

**和 Chat 的区别**：

- Chat 写新代码比较顺，Inline Edit 改已有代码更快
- Inline Edit 只看选中的代码 + 周边一小段上下文；Chat 能看更多
- **批量小改动用 Inline Edit，方案讨论用 Chat**

---

## 3. 代码补全（Tab）

Cursor 的 Tab 补全比 Copilot 激进 —— 它预测的往往是**一整段**而不是一行。

```text
你输入：func validate
AI 补全：Email(_ email: String) -> Bool {
    let regex = #"^[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}$"#
    return email.range(of: regex, options: .regularExpression) != nil
}

按 Tab 接受建议，按 Esc 拒绝。
```

**收发控制**：

- 嫌自动补全太激进：设置里关掉 `cursor.tabAutocomplete`
- 想增加预测准确度：完善 `.cursorrules`（见 [03-高级技巧](03-高级技巧.md)）
- **不要** Tab 就无脑接受 —— 它有时会补一段风格不一致的旧模式

---

## 4. 代码库搜索（`Cmd + Shift + L`）

**适合**：全局搜索代码、理解项目结构、查找相关实现。

```text
你：哪里使用了 User 模型？

Cursor：
找到 15 处使用：
1. UserListView.swift:12 - 用户列表视图
2. UserViewModel.swift:8 - 用户视图模型
3. UserService.swift:25 - 用户服务
...
```

它做的不是纯 grep，而是**语义搜索 + grep 组合**。所以问"哪里在处理用户登录"也能找到相关文件，即便代码里没 "login" 字样。

## 快捷键总表

| 快捷键 | 作用 | 典型场景 |
|------|------|------|
| `Cmd + L` | 打开 AI 对话侧栏 | 讨论、生成、解释 |
| `Cmd + K` | 内联编辑选中代码 | 小改、重构、加注释 |
| `Tab` | 接受代码补全建议 | 加速打字 |
| `Cmd + Shift + L` | 代码库语义搜索 | 全局找、理解结构 |
| `Cmd + I` | Composer（整项目级 AI） | 跨多文件大改 |
| `Cmd + Enter` | 在 Chat 里"深度模式"发送 | 复杂问题 |

> 📌 具体快捷键以 Cursor 当前版本设置为准，上表为常见默认。
