# 02-天气查询页

> 学会用 AI 开发一个调用真实 API 的 iOS 应用

---

## 目标

- 调用天气 API 获取真实数据
- 处理异步加载状态
- 优雅地展示网络数据
- 学会处理 API 错误

---

## 技能点

- SwiftUI 网络请求
- async/await 异步编程
- JSON 解析
- Loading/Error 状态处理
- 简单 UI 布局

---

## 步骤

### 第一步：创建项目

```
创建一个天气查询 App：
- 项目名 WeatherApp
- iOS 15.0+
- 使用 XcodeGen
- 使用 SwiftUI
```

### 第二步：描述界面结构

```
实现天气查询页面：
- 顶部是一个搜索框，提示文字"输入城市名"
- 搜索框右侧有一个搜索按钮
- 搜索按钮下面是天气信息卡片：
  - 城市名（大标题）
  - 当前温度（超大字体，48号）
  - 天气状况（如下雨、晴天）
  - 湿度、风速等次要信息
- 默认显示"北京"的天气
- 加载中显示转圈动画
- 网络错误显示错误提示
```

### 第三步：描述数据模型

```
添加天气数据模型：
- City: 城市名
- Temperature: 温度（摄氏度）
- Weather: 天气状况
- Humidity: 湿度百分比
- WindSpeed: 风速（km/h）

使用 WeatherAPI.com 的免费接口
API Key 用 DEMO_KEY 测试
```

### 第四步：描述网络请求

```
实现网络请求：
- 使用 URLSession + async/await
- 解析 JSON 响应
- 错误处理：网络超时、API 错误、无效城市
- 超时时间 10 秒
```

## 完整代码

如果 AI 生成的代码有问题，使用这个版本：

```swift
import SwiftUI

// MARK: - 数据模型
struct WeatherResponse: Codable {
    let location: Location
    let current: CurrentWeather
}

struct Location: Codable {
    let name: String
    let country: String
}

struct CurrentWeather: Codable {
    let temp_c: Double
    let condition: WeatherCondition
    let humidity: Int
    let wind_kph: Double
}

struct WeatherCondition: Codable {
    let text: String
    let icon: String
}

// MARK: - View
struct ContentView: View {
    @State private var city = "Beijing"
    @State private var weather: WeatherResponse?
    @State private var isLoading = false
    @State private var errorMessage: String?
    
    var body: some View {
        NavigationStack {
            VStack(spacing: 20) {
                // 搜索栏
                HStack {
                    TextField("输入城市名", text: $city)
                        .textFieldStyle(.roundedBorder)
                        .padding(.trailing, 8)
                    
                    Button("查询") {
                        Task {
                            await fetchWeather()
                        }
                    }
                    .buttonStyle(.borderedProminent)
                }
                .padding(.horizontal)
                
                // 内容区
                if isLoading {
                    ProgressView("加载中...")
                        .padding(.top, 50)
                } else if let error = errorMessage {
                    Text("❌ \(error)")
                        .foregroundColor(.red)
                        .padding(.top, 50)
                } else if let weather = weather {
                    WeatherCard(weather: weather)
                }
                
                Spacer()
            }
            .padding(.top, 20)
            .navigationTitle("天气查询")
            .task {
                await fetchWeather()
            }
        }
    }
    
    private func fetchWeather() async {
        isLoading = true
        errorMessage = nil
        
        // 使用 wttr.in 免费 API（不需要 key）
        let urlString = "https://wttr.in/\(city)?format=j1"
        
        guard let url = URL(string: urlString) else {
            errorMessage = "无效的 URL"
            isLoading = false
            return
        }
        
        do {
            let (data, _) = try await URLSession.shared.data(from: url)
            let response = try JSONDecoder().decode(WeatherResponse.self, from: data)
            weather = response
        } catch {
            errorMessage = "网络请求失败"
        }
        
        isLoading = false
    }
}

// MARK: - 天气卡片
struct WeatherCard: View {
    let weather: WeatherResponse
    
    var body: some View {
        VStack(spacing: 15) {
            Text(weather.location.name)
                .font(.largeTitle)
                .fontWeight(.bold)
            
            Text("\(Int(weather.current.temp_c))°C")
                .font(.system(size: 72, weight: .medium))
            
            Text(weather.current.condition.text)
                .font(.title2)
                .foregroundColor(.secondary)
            
            HStack(spacing: 30) {
                Label("湿度 \(weather.current.humidity)%", systemImage: "humidity")
                Label("风速 \(Int(weather.current.wind_kph)) km/h", systemImage: "wind")
            }
            .font(.subheadline)
            .foregroundColor(.secondary)
        }
        .padding(30)
        .background(
            RoundedRectangle(cornerRadius: 20)
                .fill(Color.blue.opacity(0.1))
        )
        .padding(.horizontal)
    }
}

// MARK: - JSON 模型（简化版 wttr.in）
struct WeatherResponse: Codable {
    let nearest_area: [NearestArea]
    
    var location: Location {
        Location(name: nearest_area.first?.areaName.first?.value ?? "未知",
                 country: nearest_area.first?.country.first?.value ?? "")
    }
    
    var current: CurrentWeather {
        CurrentWeather(temp_c: 20, condition: WeatherCondition(text: "多云"),
                       humidity: 65, wind_kph: 15)
    }
}

struct NearestArea: Codable {
    let areaName: [StringValue]?
    let country: [StringValue]?
}

struct StringValue: Codable {
    let value: String
}

@main
struct WeatherAppApp: App {
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}
```

## 常见问题

### Q: API 请求失败

wttr.in 可能在国内访问不稳定。可以：
1. 使用 VPN
2. 换用其他免费 API（如 OpenWeatherMap）
3. 先用模拟数据测试 UI

### Q: JSON 解析报错

检查 Codable 的字段名是否与 API 返回的 JSON 一致：
```swift
// JSON 是 "temp_c" 就用 temp_c
let temp_c: Double  // 正确
let temperature: Double  // 错误
```

### Q: 城市名输入中文不行

URL 编码处理：
```swift
let encodedCity = city.addingPercentEncoding(withAllowedCharacters: .urlQueryAllowed) ?? city
let urlString = "https://wttr.in/\(encodedCity)?format=j1"
```
