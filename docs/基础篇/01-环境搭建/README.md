## 💡 为什么需要这些工具

### Homebrew - macOS 的应用商店

想象你在手机上安装 App，你会打开 App Store。
在 macOS 上安装开发工具，Homebrew 就是那个 "App Store"。

**优势：**
- 一行命令安装软件
- 自动处理依赖关系
- 方便升级和卸载

### nvm - Node.js 版本管理器

Node.js 更新很快，不同项目可能需要不同版本。
nvm 让你可以：
- 安装多个 Node.js 版本
- 快速切换版本
- 避免权限问题

### Node.js - JavaScript 运行环境

很多 AI 编程工具都是用 JavaScript/TypeScript 写的：
- Claude Code
- Cursor 的插件系统
- Codex CLI

所以 Node.js 是必须的。

## 🗺 安装顺序

```
Homebrew
    ↓
nvm (通过 Homebrew 或 curl)
    ↓
Node.js (通过 nvm)
    ↓
Git (macOS 自带，但建议更新)
    ↓
环境验证
```

## ❓ 常见问题

### Q: 我已经安装了一些工具，怎么办？
**A:** 没关系，可以跳过已安装的部分。但建议检查版本是否最新。

### Q: 安装过程中报错怎么办？
**A:** 每个安装步骤都有"常见问题"部分，先查阅。如果还是不行，可以提 Issue。

### Q: 能不能跳过环境搭建，直接用工具？
**A:** 不建议。环境问题是最常见的坑，后面会更麻烦。花点时间一次搞定，省心很久。
