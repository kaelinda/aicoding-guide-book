| **o3-mini** | 推理轻量 | 200K | 比o1便宜93%，Structured Reasoning | $0.55~4.6/M tok |

## 二、2025~2026年模型技术趋势

### 1. 推理模型爆发（o3 / R1 / 国产推理）

推理模型（Reasoning Model）在2024年底~2025年初爆发：

```
传统模型：输入 → 直接输出（快但推理弱）
推理模型：输入 → 内部思考链 → 输出（慢但推理强）
```

- **o3**（OpenAI）：AGI Score 从"接近AGI"到"2/5"，数学/科学/代码达到人类专家
- **DeepSeek R1**（中国）：开源可本地，性能对标 o1，蒸馏版可在 Mac 上跑
- **Kimi 2**（月之暗面）：国产推理代表

**适用场景**：复杂推理、算法题、代码debug、多步骤任务。

### 2. 多模态成为标配

2026年，"纯文字模型"已成历史：

```
GPT-4o / Claude Sonnet 4 / Gemini 2.0 Flash
    ↓ 原生支持 ↓
文本 + 图像 + 音频 + 视频 + PDF + 代码截图
```

- 上传代码截图 → 模型直接分析界面/布局问题
- 上传报错截屏 → 模型理解错误上下文
- 上传设计稿 → 模型生成代码实现

### 3. 上下文窗口军备竞赛趋缓

从 32K → 128K → 1M → 200万字，现在趋于理性：

| 需求 | 推荐长度 | 代表模型 |
|------|----------|----------|
| 对话/简单任务 | 32K | GPT-4o mini, Qwen2.5 |
| 复杂代码分析 | 128K~200K | Claude Sonnet 4, GPT-4.5 |
| 代码库分析 | 500K~1M | Gemini 1.5 Pro, Kimi 2 |
| 超长文档 | 1M+ | Gemini 1.5 Pro (2M), Kimi 2 (200万字) |

> 💡 **实际建议**：不是越长越好。128K以上后，模型对远处内容的注意力会衰减。分块检索（RAG）依然是处理超长文档的主流方案。

### 4. 开源模型崛起

2024年起，开源模型快速追赶闭源：

| 闭源 | 开源对标 | 差距 |
|------|----------|------|
| GPT-4o | Qwen2.5-72B, DeepSeek V3 | < 10% |
| Claude Sonnet | Qwen2.5-Coder-32B | < 15% |
| o1 | DeepSeek R1-Distill-Qwen | < 20% |

**本地可运行最强开源编码模型**：
- **DeepSeek R1-Distill-Qwen-32B**：消费级 GPU 可跑（Mac M3 Pro ~30B）
- **Qwen2.5-Coder-32B**：HuggingFace 下载量最高的代码模型之一
- **CodeQwen1.5-7B**：轻量本地代码补全

### 5. 价格战持续

模型 API 价格持续下探，2024年降价潮：

```
GPT-4: $30/MTok → GPT-4o: $2.5/MTok（降幅90%+）
Claude Opus: $75/MTok → $15/MTok（Claude 4）
DeepSeek: $0.14/MTok（输入），比GPT-4o便宜17倍
```

## 三、模型选择指南

### 按使用场景

| 场景 | 推荐模型 | 理由 |
|------|----------|------|
| **日常编程/代码补全** | Claude Sonnet 4, GPT-4o mini | 快速、便宜、质量够用 |
| **复杂项目/架构设计** | Claude Opus 4, GPT-4.5 | 推理能力强，上下文大 |
| **代码 debug/算法** | o3, DeepSeek R1 | 推理模型，复杂分析强 |
| **中文内容创作** | Qwen2.5-72B, GLM-4.7 | 中文能力强 |
| **长代码库分析** | Claude Sonnet 4 (200K), Kimi 2 | 超长上下文 |
| **低成本批量处理** | GPT-4o mini, Gemini Flash | 价格极低 |
| **本地离线** | DeepSeek R1-Distill, Qwen2.5 | 开源可私有化部署 |

### 按硬件条件

| 硬件 | 可跑模型 | 说明 |
|------|----------|------|
| Mac M1/M2 (16GB) | Qwen2.5-7B, CodeQwen1.5-7B | 日常补全够用 |
| Mac M3 Pro (36GB) | Qwen2.5-32B, DeepSeek R1-Distill-32B | 编程主力 |
| Mac M4 Max (128GB) | Qwen2.5-72B, DeepSeek V3 (量化) | 开源旗舰 |
| NVIDIA A100 (80GB) | 全系列开源 + GPT-4o API | 生产级 |
