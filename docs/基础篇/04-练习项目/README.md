## 学习建议

### 新手推荐顺序

1. 📖 先通读需求
2. 👀 看 Claude Code 怎么拆解任务
3. 👀 观察生成的代码结构
4. ✅ 自己运行验证
5. 💬 有问题就问 Claude Code

### 遇到问题怎么办

```
❌ 报错 → 把错误贴给 Claude Code："这个错误怎么解决？"
❌ 逻辑不对 → 描述预期："应该是点击后数字加1，但现在没反应"
❌ 不知道怎么做 → 直接问："我想实现 XX 功能，应该怎么做？"
```
