## 为什么要用模板

- ✅ 结构清晰，AI 更容易理解
- ✅ 覆盖完整，不会漏掉重要信息
- ✅ 效果更好，避免模糊指令

## 使用方法

### 1. 选择模板

根据场景选择对应的模板

### 2. 填充内容

把模板中的占位符替换成实际内容

### 3. 发送给 AI

直接粘贴到 Claude Code

---

## 模板结构

```
📋 模板名称
├── 背景（为什么要做）
├── 目标（要达成什么）
├── 具体需求（做什么）
├── 约束条件（不能做什么）
├── 验收标准（怎样算完成）
└── 参考资料（相关文档）
```
