# Code of Conduct

本项目致力于营造友好、包容、尊重的协作环境。

## 我们期望的行为
- 尊重不同经验水平与观点，保持建设性沟通
- 在提出问题或建议时提供可复现信息与上下文
- 在评审时聚焦问题与改进方案，避免人身攻击

## 不可接受的行为
- 侮辱、骚扰、歧视或人身攻击
- 恶意刷屏、引战、发布与项目无关的广告或推广
- 未经许可披露他人隐私或敏感信息

## 维护者处理方式
维护者可对违反行为准则的内容采取措施，包括但不限于：
- 删除或编辑评论
- 锁定讨论
- 暂时或永久限制参与

## 举报
如遇到不当行为，请通过 Issue（避免公开敏感细节）或 Discussions 联系维护者。
